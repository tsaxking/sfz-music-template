sfzMusic: Institutions

A quick summary:

An institution is basically a big account that is accessible from all accounts that are part of the institution. This means that you can have a shared Library, Lessons, and Ensemble Management for all accounts! This is useful for schools, music studios, and other institutions that require multiple members.

Get Started

What Do Institutions Have To Offer?

Google Drive is a great resource with many benefits, but with large amounts of data it can get a little clunky. We wanted to make something similar but where each user has their own interface to interact with the same files. This means that everyone under the same institution can work on the same Library/Ensemble but they won't be getting in each other's way while they work.

Also, while Google Drive does have the ability to create roles for each user, it is not very customizable. With sfzMusic, you can create roles for each user and customize what they can and can't do with a similar workflow to Discord!

Future Features of Institutions

  • Calendar

    We are currently working on a feature that will allow you to create a shared calendar for all members of the institution. This will allow you to schedule lessons, rehearsals, and other events for all members of the institution. This will be a great way to keep everyone on the same page!

  • Communication

    While you work on a project, we want to be able to communicate easily with other members of the institution. We are currently working on a feature that will allow you to send messages to other members of the institution, create comments on specific parts of a project, and more!
